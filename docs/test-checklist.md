# 测试检查清单

用于每个需求开发完成后的验证。按实际改动范围勾选，不涉及的项目标记为“不涉及”。

## 后端

- [ ] 接口路径符合项目 API 约定。
- [ ] 只使用 GET / POST，未新增 PUT / DELETE。
- [ ] 字面量路由位于参数化路由之前。
- [ ] 请求参数校验正确。
- [ ] 响应结构符合前端使用字段。
- [ ] service 层包含业务逻辑，router 不直接写业务逻辑。
- [ ] repository 只做数据访问，不引用 HTTP 层。
- [ ] model 未使用 ForeignKey。
- [ ] 异常情况有明确处理。
- [ ] 相关后端测试已新增或更新。
- [ ] 后端测试已运行并记录结果。

## 前端

- [ ] `frontend/api/` 封装与后端 method/path/request/response 一致。
- [ ] 页面成功状态正常。
- [ ] 页面加载状态正常。
- [ ] 页面空状态正常。
- [ ] 页面错误状态正常。
- [ ] 用户重复点击、空输入、网络错误等边界情况已处理。
- [ ] 页面根元素使用 `height: 100vh` + `overflow: hidden`（适用时）。
- [ ] `scroll-view` 在 flex 布局中设置 `flex: 1` + `height: 0`（适用时）。
- [ ] flex 子元素设置 `width: 100%` + `box-sizing: border-box`（适用时）。
- [ ] 无横向滚动或异常页面级滚动。
- [ ] 前端检查/构建命令已运行并记录结果。

## API 联调

- [ ] 前端请求 method 正确。
- [ ] 前端请求 path 正确。
- [ ] 请求体字段名和后端 schema 一致。
- [ ] 响应字段名和页面渲染字段一致。
- [ ] 错误响应能被前端正确提示。
- [ ] 列表分页字段正确。
- [ ] 时间范围字段正确（如涉及）。

## 回归主流程

- [ ] 打开首页正常。
- [ ] 待办列表正常显示。
- [ ] 创建待办正常。
- [ ] 查看详情正常。
- [ ] 修改待办正常。
- [ ] 删除待办正常。
- [ ] 聊天/智能创建入口不受影响。
- [ ] 语音创建入口不受影响。
